Project Overview 
  This Project analyzes Amazon product reviews using Microsoft Excel to uncover customer sentiment, rating distribution, and review trends.

Tools Used
  Microsoft Excel
  Pivot Table
  Charts
  Data Cleaning Techniques

Insights
  1. Average rating analysis
  2. Percentage of 4* review
  3. Review distribution patterns
  4. Most active reviewers

Dataset Source 
 UCSD Amazon Review Dataset
